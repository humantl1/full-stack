Git Guide

This is a work-in-progress simple command guide to using command line git.




Command Description Resources
Initialization git init Initialize a .git directory
git init -b master Initialize local directory as a git repository
git remote add origin 'remote URL' Set remote URL as the master repository
Committing git add 'filename' Stage file changes
git add . Stage all files
git commit -m 'description' Commit files
git push origin master Push changes to the master repository
Miscellaneous git status Display file modification and commit status
git log Display commit history
git reset --hard commit-id Delete all changes after specified commit ID
